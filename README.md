# 🚀 GenAI Job Preparation Platform

An AI-powered full-stack web application that helps users prepare for jobs smarter, not harder.

Upload your resume or describe your profile, paste a job description, and get a **personalized interview preparation strategy**, including:

- 🎯 Match Score for the role  
- 🧠 Technical & Behavioral Questions  
- 🗺️ Step-by-step Preparation Roadmap  

---

## ✨ Features

- 📄 Upload Resume (PDF/DOCX) or add self-description  
- 🧠 AI-generated Technical & Behavioral Questions  
- 📊 Match Score based on job requirements  
- 🗺️ Personalized preparation roadmap  
- 📥 Download AI-generated resume (PDF)  
- ⚡ Clean and responsive UI  

---

## 🛠️ Tech Stack

### Frontend
- React.js
- SCSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### AI Integration
- Google Gemini API

---

## ⚙️ How It Works

1. Paste a job description  
2. Upload resume or enter your skills  
3. AI analyzes your profile  
4. Generates:
   - Match Score  
   - Interview Questions  
   - Preparation Plan
